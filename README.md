## Prerequisites

- node ^14 || ^15 || ^16
- npm please-use-yarn
- yarn >= 1.19.1

## Install

```sh
yarn install
```

## Usage

### Serve Frontend
```sh
yarn serve
```

### Start UI lib
```sh
yarn storybook:ui
```

## Run tests

```sh
yarn test
```
